# eazycv
EazyCV is an AI-powered resume generator that helps users quickly create professional, personalized CVs with smart suggestions and modern templates.

## Project structure

```text
eazycv/
├─ cv.html          # Generated CV view (renders the resume)
├─ form.html        # Form where users enter their information
├─ index.html       # Landing/home page (optional)
├─ login.html       # Login page (optional)
├─ js/
│  ├─ form.js       # Handles form submission and saves data
│  ├─ cv.js         # Reads saved data and fills the CV
│  ├─ home.js       # Scripts for the home page
│  ├─ login.js      # Scripts for the login page
│  └─ helpers.js    # Shared helper functions
├─ styles/
│  ├─ form.css      # Styles for the form page
│  ├─ cv.css        # Styles for the CV page
│  ├─ global.css    # Global/shared styles
│  ├─ home.css      # Styles for the home page
│  └─ login.css     # Styles for the login page
├─ images/          # Image assets
├─ LICENSE
└─ README.md
```
